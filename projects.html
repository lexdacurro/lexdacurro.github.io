Water Use Site/Wells Information Retrieval Web Application for United States Geological Survey - Lower Mississippi Gulf Water Science Center at Little Rock Arkansas.

The first ever Water Use App is a web application that retrieves wells information from different wells/sites in Arkansas. It allows the public such as agencies like ANRC (Arkansas National Resource Commission), local/state agencies and as well as local farmers to retrieve necessary Water Use information. The main feature of this project is to be able to retrieve Water Use site-specific and the aggregated data for each Site/Well. It also features generating reports in formats like PDF, CSV and TSV.
  • Date: May 2015 - Ongoing (Currently in Beta Phase)
  • Back-end: OOP PHP => 5.5
  • Client-side: HTML, CSS3, jQuery
  • Database: Oracle
  • Data format: JSON/XML
